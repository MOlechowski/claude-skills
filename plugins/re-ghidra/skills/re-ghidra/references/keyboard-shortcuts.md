# Ghidra Keyboard Shortcuts

## Navigation

| Shortcut | Action |
|----------|--------|
| G | Go to address/label |
| Ctrl+E | Go to entry point |
| Alt+Left | Back |
| Alt+Right | Forward |
| Ctrl+Shift+E | Set selection end |

## Search

| Shortcut | Action |
|----------|--------|
| Ctrl+Shift+E | Search memory |
| S | Search for strings |
| Ctrl+F | Find (in listing) |
| Ctrl+Shift+F | Find references to |

## Editing

| Shortcut | Action |
|----------|--------|
| L | Rename (label/variable) |
| F | Edit function signature |
| ; | Set EOL comment |
| Ctrl+; | Set pre-comment |
| Shift+; | Set post-comment |
| T | Set data type |
| C | Disassemble (create code) |
| D | Create data |
| U | Undefine |
| P | Create pointer |
| ' | Create char/string |

## Selection

| Shortcut | Action |
|----------|--------|
| Ctrl+A | Select all |
| Shift+Click | Extend selection |
| [ | Select to start of function |
| ] | Select to end of function |

## View

| Shortcut | Action |
|----------|--------|
| Space | Toggle listing/decompiler |
| Ctrl+E | Show symbol references |
| Ctrl+Shift+B | Show bytes |

## Analysis

| Shortcut | Action |
|----------|--------|
| A | Auto-analyze |
| F | Create function |
| Ctrl+Shift+G | Compute checksums |

## Project

| Shortcut | Action |
|----------|--------|
| Ctrl+S | Save |
| Ctrl+Z | Undo |
| Ctrl+Shift+Z | Redo |

## Decompiler Window

| Shortcut | Action |
|----------|--------|
| L | Rename variable |
| Ctrl+L | Retype variable |
| ; | Set comment |
| Click + G | Go to definition |

## Function Graph

| Shortcut | Action |
|----------|--------|
| + | Zoom in |
| - | Zoom out |
| 0 | Fit to window |
| Home | Go to entry block |

## Tips

- Most shortcuts work in Listing view
- Decompiler has subset of shortcuts
- Right-click shows available actions with shortcuts
- Edit > Tool Options > Key Bindings to customize
